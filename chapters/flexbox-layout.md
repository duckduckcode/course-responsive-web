# FlexBox Layout

Flexbox is best for arranging elements in either a single row, or a single column.

## Navigation Bar

```
.main-nav {
  display: flex;
  flex-direction: row;
}

.main-nav a {
  width: 200px;
}

.main-nav .title {
  flex: 1;
}
```

Your title should be on the left, and links on the right.
